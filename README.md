# zh-loading
> 绚丽 vue loading 组件
## npm adress
`https://www.npmjs.com/package/zh-loading`

## 1. Install
  ```
  npm: npm install zh-loading
  ```
  ```
  yarn: yarn add zh-loading
  ```

## 2. Quick Start
1.引入 `zh-loading` 组件
```
import ZhLoading from 'zh-loading';
```

2.在vue组件中使用
```
<template>
  <div>
    <ZhLoading />
  </div>
</template>
```
